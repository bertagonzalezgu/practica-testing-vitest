# Ejercicio de Testing - Calculadora

## Objetivo

El objetivo de este ejercicio es practicar la escritura de tests unitarios con **Vitest**.

Actualmente la calculadora solo tiene implementada la función `sum()`. Tu tarea será completar el resto de operaciones con sus tests y hacer que todos los tests pasen correctamente.

---

## Tareas

### 1. Completa la calculadora

En el archivo `calculator.ts` deberás implementar las siguientes funciones:

- `subtract(number1, number2)`
- `multiply(number1, number2)`
- `divide(number1, number2)`

La función `sum()` ya está creada como ejemplo.

---

### 2. Completa los tests

El proyecto ya incluye un archivo de tests con una prueba para la función `sum()`.

Utilizando ese test como referencia, deberás añadir las pruebas necesarias para verificar el correcto funcionamiento de las funciones que implementes

Tu objetivo es:

- Implementar las funciones que falten
- Importarlas en el archivo de test.
- Crear los tests para las nuevas operaciones.
- Comprobar que cada función devuelve el resultado esperado.
- Verificar que todos los tests pasan correctamente.
